# sysprims v0.1.2

**Release Date:** 2026-01-19
**Status:** Security & CI/CD Maintenance Release

## Summary

Security patch addressing a high-severity vulnerability in CI/CD dependencies, plus infrastructure improvements for release workflow clarity and dependency vulnerability scanning.

## Security

### GHSA-cxww-7g56-2vh6 (High)

**Component:** `actions/download-artifact`
**Fixed:** Updated from `@v4` to `@v4.1.3`

The GitHub Actions `download-artifact` action had a path traversal vulnerability that could allow artifact downloads to write outside the intended directory. This affected our release workflow which downloads build artifacts for signing.

**Impact:** CI/CD pipeline only. No impact on sysprims library code or released binaries.

**Mitigation:** Update to v4.1.3 which includes the fix (available since 2025-09-04).

## Verification

Verify this release with the signed checksums:

```bash
# Download release and verification files
curl -LO https://github.com/3leaps/sysprims/releases/download/v0.1.2/SHA256SUMS
curl -LO https://github.com/3leaps/sysprims/releases/download/v0.1.2/SHA256SUMS.minisig
curl -LO https://github.com/3leaps/sysprims/releases/download/v0.1.2/sysprims-minisign.pub

# Verify signature
minisign -Vm SHA256SUMS -p sysprims-minisign.pub

# Verify checksums
shasum -a 256 -c SHA256SUMS --ignore-missing
```
